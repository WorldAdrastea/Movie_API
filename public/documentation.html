Movie API Documentation Page

  • 1. Return a list of ALL movies to the user
  • 2. Return data (description, genre, director, image URL, whether it's featured or not) about a single movie by title to the user
  • 3. Return data about a genre (description) by name/title
  • 4. Return data about a director (bio, birth year, death year) by name
  • 5. Allows users to register
  • 6. Allows users to update their info
  • 7. Allow users to add a movie to their list of favourites
  • 8. Allow users to remove a movie from their list of favourites
  • 9. Allow users to unregister.

Using the format of Business Logic - URL - HTTP Method - Parameters - Request body format - Respond body format

1. Return a list of ALL movies to the user - /movies - GET - N/A - N/A - JSON Object

2. Return data by title - /movies/:title - GET - :title - N/A - JSON Object

3. Return data by genre - /movies/genre/:genreName - GET - :genreName - N/A - JSON Object

4. Return data by director - /movies/director/:directorName - GET - :directorName - N/A - JSON Object

5. Allows new users to register - /users - POST - N/A - JSON Object

6. Allows users to update their info - /users/:id - PUT - :id - N/A - JSON Object

7. Allows users to add a movie to their list of favourites - /users/:id/:movieTitle - POST - :id :movieTitle - N/A - JSON Object

8. Allows users to remove a movie from their list of favourites - /users/:id/:movieTitle - DELETE - :id :movieTitle - N/A - JSON Object

9. Allow users to unregister - /users/:id - DELETE - :id - N/A - It will return: 'User [name] has been deleted'
